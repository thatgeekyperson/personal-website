# Personal Website

A minimal, production-ready personal website built with React, Vite, TypeScript, and Tailwind CSS v4.

## Quick Start
1. `npm install`
2. `npm run dev`

## Documentation (SSoT)
This repository follows the **Hub and Spoke** documentation model. 

- **AGENT.md**: The Single Source of Truth for architecture, tech stack, and logic.
- **plans/**: Architecture Decision Records (ADRs) for each development phase.

## Tech Stack Highlights
- **Vite 8** + **React 19**
- **Tailwind CSS v4**
- **TypeScript**
- **Vitest** for testing

For full technical specifications, constraints, and pipeline logic, refer to [AGENT.md](./AGENT.md).
